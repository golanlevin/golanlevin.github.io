Demo: Painting with Interactive Pixels

This Java demo was presented in a talk given Jan. 23. 1998 for the Stanford seminar CS547: Seminar on People, Computers, and Design .

The talk was taped and can be seen live from here until mid february.

The idea with this editor is to show that it it possible to construct design tools where the basic building blocks have inherent behavior. This is a very early prototype with only two tools implemented: Pen and Cursor.

You use the pen to paint as in any paint editor. Try using the arrows and cross "inks", i.e. "interactive" inks.

When you try clicking on the painting with the cursor tool you will find the you have created an interactive painting.
